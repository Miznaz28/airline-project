## *# ✈️ Airline Management System*



### *## 📌 Overview*



*This is a simple Airline Management System built using Python Flask and SQLite.*



*It allows users to:*



*\* Register and login*

*\* View available flights*

*\* Book tickets*



*Admin can:*



*\* Add new flights*

*\* Manage flight data*



*---*



### *## 🛠️ Tech Stack*



*\* Backend: Python (Flask)*

*\* Frontend: HTML*

*\* Database: SQLite*



*---*



### *## 🚀 Features*



*\* User authentication*

*\* Flight booking system*

*\* Admin panel*

*\* Database integration*



*---*



### *## ▶️ How to Run*



*```bash*

*pip install flask*

*python app.py*

*```*



*Open in browser:*



*```*

*http://127.0.0.1:5000*

*```*
